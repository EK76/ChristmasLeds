# ChristmasLeds
Christmas Leds is Arduino project, whose RGB leds can be uses for example as a Christmas decorative lighting. which the project name refers to.
The code is written by c++ programming language with Arduino IDE complitator. You can find more info about Arduino IDE from this link
https://www.arduino.cc/en/software
